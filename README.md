# backend-node

## pre requisite
- install [Node.JS](https://nodejs.org/en/download/)
- install nvm
- install npm
- run `npm install`

## copy environtment file
`cp .env.example .env`

## Run migration file
`knex migrate:latest`

## start at development
`npm run dev`

## start the server
`npm run start`


please check `Test Kemenkes - Parking.postman_collection.json` at your root folder for api documentation
